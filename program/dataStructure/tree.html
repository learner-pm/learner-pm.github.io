# 递归遍历

通过递归来进行遍历 tree。 按照访问value值得位置不同，分为前序遍历。

const data = (obj) => {
  if (obj === null) return;
  else {
    console.log(obj.value);
    data(obj.left);
    data(obj.right);
  }
};

# 层次遍历

层次遍历就是一层一层得去遍历 tree。

通过队列来进行储存来访问节点。

const data = (obj) => {
  const arr = []; // 队列来储存节点
  arr.push(obj);
  while (arr.length !== 0) {
    if (arr[0].left) {
      arr.push(arr[0].left); //入队
    }
    if (arr[0].right) {
      arr.push(arr[0].right);
    }
    console.log(arr[0].value);
    arr.shift(); //出队
  }
};
